File Watching

Gulp watches our files for changes.

We can choose specific files, or whole groups with wildcards



    scripts: [ // Application javascripts
      '!app/docs/**', // ignore documentation in app
      '!app/bower_components/**', // ignore bower-ingested scripts
      '!app/**/*_test.js', // ignore our test scripts
      '!app/scripts/templates.js', // template script
      '!app/_gulp/**', // ignore all files in _gulp directory
      'app/**/*.js'], // main application file
