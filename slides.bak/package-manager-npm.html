Package Manager: NPM

Node Package Manager

  • Installs NodeJS packages
  • Small development scripts


Example Packages:

  • GH Pages
  • Uglify
  • Chalk
* preferred to be single-action software (general rule) * all the little things that we could write ourselves...but just don't have to anymore! * GH Pages: builds in the myriad commands to switch branch, add/commit, push to server, switch back * Uglify: minification package * Chalk: *tiny example* colors messages in terminal
