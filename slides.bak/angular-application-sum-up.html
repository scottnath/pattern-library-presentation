Angular Sum-Up

  • Dependency injection
    = stackable, modularized functionality
     
  • Built with testability in mind
    = consistent code that's proven to be functional
     
  • Massive community
    = free work & tons of solutions
     
  • Efficient and fast for end-users
    = happy customers
     
  • Can be used on non-AngularJS projects
    = can roll out new pieces on existing sites
